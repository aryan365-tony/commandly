# **App Name**: Control Canvas

## Core Features:

- Layout Manager: Display a list of locally saved control layouts with options to edit, delete, or open them.
- Layout Builder: Drag-and-drop interface to create custom control layouts with buttons, joysticks, sliders and D-pads, including editable properties like label, size, position and output index.
- Communication Settings: Configure communication methods (Bluetooth Classic/BLE, WiFi TCP/UDP, MQTT, WebSocket, USB Serial) with protocol-specific configurations and data sending intervals, and store the config alongside layouts.
- Controller Runtime: Render the selected control layout and transmit component states as an array in real-time using chosen protocol, offering optional full-screen joystick and debug log.
- Full-Screen Joystick: Display a full-screen joystick for fine-grained control.

## Style Guidelines:

- Primary color: Teal (#008080) for a modern, tech-focused feel.
- Secondary colors: Light gray (#D3D3D3) and dark gray (#A9A9A9) for a clean and neutral background.
- Accent: Electric Blue (#7DF9FF) for interactive elements and highlights.
- Clean and modern sans-serif typeface for labels and UI elements.
- Use simple and consistent icons for layout actions (edit, delete, open), communication methods, and control components.
- Consistent spacing and alignment to provide a structured and intuitive user experience across all screens.
- Subtle animations for control state changes and screen transitions, such as button presses and layout switches.
